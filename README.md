# glovepunk

Custom keymap configuration for the MoErgo Glove80 keyboard.

## Features

- Custom Enthium layout with home row modifiers
- Media controls on top row (previous, volume down, volume up, next)
- Simplified 11-layer structure
- Custom key mappings optimized for personal workflow

## Layers

0. Enthium (Base)
1. Typing
2. Cursor
3. Number
4. Function
5. Symbol
6. System
7. Gaming
8. Factory
9. Lower
10. Magic
